-----
タイトル： Toilet God（トイレの神様）
チーム名：トイレクリーナー

このゲームは、Global Game Jam 2016で作られました。
会場：岡山会場
-----
■インストールについて
　特にありません。マウスをご用意ください。


■ゲーム紹介
　あなたの朝の生活「習慣」といえば何でしょうか。トイレ？いいですね！
　このゲームは、汚くなったトイレを綺麗に磨いて、萌えなトイレの神様を救うゲームです。


■操作方法
　一人プレイ。
　マウスのクリック＆ドラッグゲームです。
　スプレー剤の制限回数は5回までです。
　ステージが進むと雑菌の量が増えるので、綺麗に磨きましょう！


■プラットフォーム
　Windows
